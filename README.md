# Nies Puck – Website (statisch, lokal testbar)

## Lokal ansehen
- Datei `index.html` öffnen **oder** (empfohlen):
- Im Ordner ein Terminal öffnen und starten:
  - `python -m http.server 8000`
  - im Browser: `http://localhost:8000`
